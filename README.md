# Usage

html

```html
<link rel="stylesheet" href="@/pmd.css" />
...
<button id="sample-button" class="pmd-button pmd-ripple"></button>
```

js

```javascript
import * as pmd from '@/pmd.js';

const ripple = new pmd.ripple(document.getElementById('sample-button'));
```
